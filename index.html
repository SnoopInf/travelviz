TravelViz

Various travel visualizations for Ukraine

Download this project as a .zip file Download this project as a tar.gz file


In ideal world, this chart should show interest and availability of travelling from Ukraine to each country.

I'm still working on it. Size of the bubble represents travel cost, distance from Ukraine to country bubble represents flight duration. Color is visa info.

Data visualization course project by Kirill Davidenko. I've taken data about visa for Ukraine here. Also i've used this tool to convert CSV to JSON
